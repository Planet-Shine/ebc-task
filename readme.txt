
Как смотреть результат.

npm, nodejs должны быть установлены.

Клонировать репо https://github.com/PavelOfficial/ebc-task.git.
Выполнить в терминале: npm install.
Выполнить сборку: webpack. (Или "node ./node_modules/webpack/bin/webpack.js")
Запустить сервер npm run start.
Перейти в браузере по адресу http://127.0.0.1:8000/.
